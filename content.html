Лекция 1 - переменные

Содержание видео

Локальные и глобальные переменные

00:10 — Глобальные переменные.

00:30 — Объявление глобальной переменной.

01:40 — Определение: глобальная переменная — к которой имеют доступ разные методы.

01:50 — Замена глобальных переменных на локальные.

02:50 — Минусы глобальных переменных. В больших проектах сложно отследить изменения глобальных переменных, это может привести к ошибкам.

04:10 — Значение глобальных переменных по умолчанию.

06:40 — Область видимости переменных.

09:25 — “эмпирическое” правило видимости переменных. Они видны внутри { } Ошибка в содержании? Предложите исправление!

Заметки по лекции

Переменная доступна (грубо) внутри тех фигурных скобок, где она определена. Область кода, из которой допустимо обращение к переменной, называется "областью видимости". Локальная переменная "перекрывает" глобальную с тем же именем. Однако локальная переменная не может перекрыть другую локальную переменную с тем же именем — возникнет ошибка компиляции.
